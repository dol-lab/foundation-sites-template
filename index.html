Scroll Snap

Basics

The Scroll Snap Container is an overflow-container that contains elements arranged horizontally (or vertically) and when scrolled, those elements “snap” into place.

You give a .scroll-snap-container-x class to the parent and a .scroll-snap-element to the children.

Snapping

To tell the elements where to snap to give the .scroll-snap-container-x one of the following classes:

  • .snap-align-start
  • .snap-align-center
  • .snap-align-end

Width

By default, the width of the .scroll-snap-element is 100%. There is, however, room for variation. Apply one of the following classes to the .scroll-snap-container-x.

  • .width-fixed
  • .width-small
  • .width-medium
  • .width-large

It is set-up like so:

Advanced

Breakpoints

For specific breakpoints and upwards add one of the following to .scroll-snap-container-x:

  • .small-scroll-snap
  • .medium-scroll-snap
  • .large-scroll-snap
  • .xlarge-scroll-snap
  • .xxlarge-scroll-snap

To remove scroll-snapping for specific breakpoints and upwards add one of the following to .scroll-snap-container-x:

  • .small-no-scroll-snap
  • .medium-no-scroll-snap
  • .large-no-scroll-snap
  • .xlarge-no-scroll-snap
  • .xxlarge-no-scroll-snap

Tabs

Tabs are elements that help you organize and navigate multiple documents in a single container. They can be used for switching between items in the container.

Basics

There are two pieces to a tabbed interface: the tabs themselves, and the content for each tab. The tabs are an element with the class .tabs, and each item has the class .tabs-title. Each tab contains a link to a tab. The href of each link should match the ID of a tab. Alternatively, the ID can be specified with the attribute data-tabs-target.

1
2
3
4
<ul class="tabs" data-tabs id="example-tabs">
  <li class="tabs-title is-active"><a href="#panel1" aria-selected="true">Tab 1</a></li>
  <li class="tabs-title"><a data-tabs-target="panel2" href="#panel2">Tab 2</a></li>
</ul>

The tab content container has the class .tabs-content, while each section has the class .tabs-panel. Each content pane also has a unique ID, which is targeted by a link in the tabstrip.

Icons and Long Titles

Add a span-Tag around the title within the link this way long titles will be truncated.

You can also add Icons to your tabs. When you do, make sure you wrap your label in a span-Tag.
